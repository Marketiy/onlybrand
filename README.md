# Статический сайт (базовая структура)

Проект для простого многостраничного сайта без сервера и без фреймворков.

## Структура

- `index.html` — главная страница  
- `about.html` — страница «О нас»  
- `services.html` — страница «Услуги»  
- `contacts.html` — страница «Контакты»  
- `portfolio.html` — страница «Портфолио»  
- `assets/css/styles.css` — основные стили  
- `assets/js/main.js` — основной скрипт  
- `assets/libs/jquery/jquery.min.js` — локальная копия jQuery  
- `assets/fonts/` — локальные шрифты

## Зависимости

- jQuery (локально, файл `assets/libs/jquery/jquery.min.js`)
